# 8.5 Family Travel Tracker

## Overview

The **8.5 Family Travel Tracker** is a web application designed to help families collaboratively plan and track their travels. It allows users to create accounts, add family members, and manage travel itineraries with ease.

## Features

- **User Authentication**: Secure sign-up and login functionality.
- **Family Management**: Add and manage family members within the application.
- **Travel Itineraries**: Create, view, and edit travel plans for the family.
- **Responsive Design**: Accessible on various devices, ensuring a seamless experience.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: Passport.js

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/melvinprince/8.5-Family-Travel-Tracker.git
   cd 8.5-Family-Travel-Tracker
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Set Up Environment Variables**:

   Create a `.env` file in the root directory and add the following variables:

   ```env
   PORT=3000
   MONGODB_URI=your_mongodb_connection_string
   SESSION_SECRET=your_session_secret
   ```

4. **Start the Application**:

   ```bash
   npm start
   ```

   The application will run on `http://localhost:3000`.

## Usage

1. **Sign Up**: Create a new account using your email and a secure password.
2. **Add Family Members**: Navigate to the family management section to add members.
3. **Create Travel Plans**: Use the itinerary feature to plan upcoming trips.
